# Overwatch Replay Analyzer (ORA) v0.1

## 需求说明

v0.1 需求列表

1. 根据用户输入读取owl直播录像视频和比赛内容，用户输入包括：
  * 视频所在文件路径
  * 当前比赛在视频中的起止时间
  * 当前比赛的队伍名称，每个队的选手ID,按视频中从左到右依次排列
  * 分析结果输出路径

2. 根据OWL直播录像，每0.5秒捕捉一帧，对当前帧识别并记录如下内容：
  * killfeed，包括：
    * 击杀者（英雄+选手）
    * 被击杀者（英雄+选手）
    * 击杀使用技能
    * 是否爆头
    * 复活者（英雄+选手）
    * 被复活者（英雄+选手）
  * 当前所有英雄状态，包括： 
    * 是否死亡
    * 大招状态（充满/未充满）
  * 对以下内容进行自动识别并从视频中排除：
    * 击杀回放
    * 游戏暂停

3. 将捕捉到的数据按标准格式记录并保存于用户指定路径，标准格式见/docs
